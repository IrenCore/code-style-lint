# create-code-style-lint

Code style lint config for front-end project.

## Scaffolding Your Eslint

### With npm

```bash

 npm create code-style-lint

```

### With yarn

```bash

 yarn create code-style-lint

```

### With pnpm

```bash

 pnpm create code-style-lint

```
